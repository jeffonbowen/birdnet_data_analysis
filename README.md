# BirdNET Detection Summary

Analysis and summary of data collected on BirdNet-Pi

-   Static dashboard using Quarto, hosted on Githib.

-   Data needs to be cleaned and some QA aspect.

-   Summary of data collected to data and date ranges. number of days per week that it has been operating and collecting data. A data collection summary for the station.

-   Species list with detection rate.

-   Detections over time:

    -   Detections per day of the year, separated by year.

    -   Species richness per calendar day, separated by year.

    -   By individual species (user selected)

-   Open to other ideas.
